About Me

My name is Brennan. I'm 16 years old, and I like computers

I started programming when I was 13. I started out with Java. I learned how to program Java fluently and efficiently in about 3 weeks, using only resources I found online in Youtube Videos, and on forums like Stack Overflow.

Later that year, I learned C and C++ for an independent school project. For this project I had a mentor, but again, most of the learning was done on my own.

Fast forward a year: I started taking Computer Science and re-learned C, and then learned another language - Objective-C. I did some OSX and iOS development for a while, but decided that the Apple development process wasn't really my style.

The following semester I worked a lot with the little Raspberry Pi Computers and Python. I didn't find Python particularly interesting, as it didn't have a syntax I really liked. I did however learn it.

After school let out, I didn't slow down at all. I kept working, and I was excited to start my independent study class in the spring of the next year.

So, to keep up with my classes, I started taking Geometry over the summer in order to fast-forward my math classes for the following year.

I also decided to learn HTML/CSS and JavaScript. Then, I started working on my site

My site is basically stolen from a bunch of code I wrote in the process of learning how to program HTML/CSS. I just repurposed most of this into useable code, and made everything very pretty. Even the page you're viewing right now comes from a Tumblr theme that I wrote over the summer.

I now run several "Art Blocks" (mini one-quarter long classes that specialize in various things) regarding technology. My favorite being the "whatever you want with tech" Artblock, where the kids can work on anything they want to as long as it's not homework.

So yeah, that's been me.

Thank you for reading!

- Brennan D. Macaig, Sant Bani School
