What Is JavaScript?

JavaScript is a scripting-based programming language. Basically, it's an easy way of typing some specialized code. It's a mixture of Java, the programming language by ORACLE, and a scripting-language that's heavily watered down for web developers to be able to easily use. JavaScript is designed to run within a website's code. So, inside the code of my website, somewhere there is a line that includes some HTML code. Inside this HTML code, there is a script tag. This tag allows a web developer to insert some JavaScript.

Why do you use JavaScript?

As I said above, JavaScript is a specialized programming language. It allows web developers to easily and very quickly add elements to a page that do special things. On my site, I use JavaScript to randomize the backgrounds on the front page, to create the menu bars in the back pages, to create the up buttons, to render certain elements of text including my emails, just about every page on my site uses JavaScript. It's very important you enable JavaScript so that you can view my site as well as possible.

What Happens Without JavaScript?

Well, if you try using my site without JavaScript anywhere other than an ERROR page or a direct-link to a blog post, then you'll get redirected to an error page. This page will basically say "Go enable JavaScript, or else you can't use my site." It's a users choice if they want to enable JavaScript at this point or not. If you choose not to, then my site will simply not be viewable. Nothing I can do about that, other than re-write my site in a non-JavaScr - Yeah, no.
