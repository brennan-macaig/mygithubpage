Projects Period

p>Projects Period is done over a two week vacation at the end of Febuary and the begining of March.

It is a 15-25 Hour long project (length depending on your grade), where the student gets to decide what they want to research.

You can choose anything you want, as long as it gets approved in a formal proposal format by the Projects Period Comitee.

Eigth Grade, Learning C/C++

In Eigth Grade, I worked on programming C and C++ for Projects Period. I was mentored by the fantastic Mr. Steve Olson, and boy did I learn a lot. I completed an entire course at Lynda.com on C++ (with some basic C thrown in), and became fluent with the base functions in C/C++. I still use my knowledge from this project to this day.

Ninth Grade, Building a Computer

In ninth grade, I built my own computer. I did around 25 hours of research on the full details of how a computer works; including full break downs on each individual part. At the end of all this, I built my very own computer that I still use as my main computer to this day. This project helped me around a lot with the hardware side of things in computer. I use this knowledge to help fix computers for people by knowing how each part functions, and what it's intended purpose is.

Tenth Grade, 3D Animation and Design

In tenth grade, I created 3D animations and models using Blender3D. I created everything from raindrops on a window to things like a roughly 1 minute long animation.
